Sage Analysis Group - Blurred Powerpoint Tool

This tool can be used to create blurred versions of powerpoints.

1). In powerpoint, select "save a copy" and change the file type from ".pptx" to ".jpg", then hit save

A pop-up will ask "Which slides do you want to export?" - select "All Slides"

Powerpoint will then create a new folder containing jpg versions of all slides from the powerpoint

2). Click the "choose files" button and highlight all of the .jpgs exported by powerpoint, then click "open"

This tool will create blurred versions of all the slides selected, sort them alphabetically, add them to a powerpoint, and start a download of that powerpoint.

It will take a moment to setup the web app, a "choose file" button will appear when everything is ready
